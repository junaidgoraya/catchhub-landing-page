# CatchHub Landing Page

A modern and responsive landing page developed using **HTML5** and **CSS3** as part of the **FentixTech Web Development Internship – Task #01**.

The objective of this project was to recreate a provided UI design as accurately as possible by focusing on layout, spacing, typography, colors, and responsiveness.
